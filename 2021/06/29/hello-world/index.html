从零开始搭建个人博客

 2021-06-29 15:06:05  
  • 折腾教程 
 
  • 个人博客 
 316 Words  1 Mins  

我们使用 Hexo 下的 keep 主题 + github pages 从零搭建一个个人博客。
同时配置自定义域名解析，并使用 github 的 actions 进行持续集成。

Quick Start

我本来是没有考虑持续集成这个功能的，因此走了不少弯路，同时网上大部分教程多少有些瑕疵，此文章着重记录下遇到的坑。

主要步骤

参考这个教程 进行项目初始化并配置CI。
注意 项目工程下的keep主题也是一个git仓库，我们如果自定义了一些主题配置的话，当我们push时并不会生效，需要把keep主题下的.git文件夹删了或者自己fork一份keep，并使用自己fork的代码。

主要的坑

按照上述步骤做完可能发现自己的主页无法显示图片甚至什么都显示不了，那是因为我们需要先主动部署一次 并把代码推送到 公共仓库的 gh-pages 分支，这样这个分支才会有网站需要的静态文件。

还有 解析自定义域名的时候需要多加几条A记录 并且一定要使用CNAME解析到自己的username.github.io主页，
